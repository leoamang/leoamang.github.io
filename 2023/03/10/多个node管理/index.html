nvm镜像源更新

imes系统的node版本需求为12.20.0v，但本机没下载，使用nvm install 12.20.0时报错：

原因是淘宝镜像源失效，所以更改淘宝镜像源后下载：

  1. 找到nvm配置文件

    nvm root

  2. 修改配置：

    node_mirror: https://npmmirror.com/mirrors/node/

    npm_mirror: https://npmmirror.com/mirrors/npm/
