git相关使用拉取命令

发表于2024-12-17|更新于2024-12-17
|阅读量:

使用的相关git命令：

克隆代码：

git clone {{仓库地址}}

拉取代码：

拉取远端分支：git pull

另一种可以通过vscode/cursor中的分支菜单选择拉取

推送代码：

将代码从共工作区到暂存区：git add .

提交暂存区到本地仓库：git commit -m {{备注}}

推送：git push

另一种可以通过vscode/cursor中的分支菜单将已经修改的文件点击“+”新增到暂存区→在输入框里加入备注并提交到本地仓库→同步修改推送到远端仓库。

相关分支：

查看分支： git branch #应该是本地的分支

查看分支： git branch -r#远程仓库里面的分支

查看分支： git branch -a#查看本地和远程仓库中的分支

创建分支： git branch {{分支名称}}

切换分支： git checkout {{分支名称}}

创建+切换分支：git checkout -b {{分支名称}}

合并某分支到当前分支： git merge {{分支名称}}

删除分支： git branch -d {{分支名称}}

更新分支：git fetch

查看仓库状态

查看变更：git dtatus
